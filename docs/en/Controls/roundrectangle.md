## 1. Overview

Round rectangle shape controls are used to create and edit rectangular graphic elements with rounded corners on the canvas. They combine the practicality of rectangles with the visual softness of rounded corners, and are widely used basic shape elements in modern interface design.

![Image Description](../../images/控件照片/圆角矩形/1.png)

2. Usage Scenarios

Round rectangle controls are used in the following scenarios:

### 1. Data Visualization Scenarios

- Used as auxiliary shapes for graphical display of data relationships
- Round rectangle elements in data distribution diagrams and relationship diagrams
- Round rectangle backgrounds for progress indicators and status identifiers

## 3. Quick Start

1. Select the round rectangle control from the toolbar
2. Place the mouse at the target position on the canvas, hold down the left mouse button
3. Drag the mouse to adjust the size and proportion of the round rectangle
4. Release the left mouse button to complete round rectangle creation
5. Adjust overall size through the 8 control points of the selection box
6. Use the rotation control point to adjust the angle direction
7. Precisely set corner radius, size, and style in the property panel

## 4. Property Details

| Name   | Description                                                                   |
| ------ | ----------------------------------------------------------------------------- |
| Name   | The name of this control.                                                         |
| X      | The distance from the left side of the control to the left side of the canvas, in px.                                     |
| Y      | The distance from the top of the control to the top of the canvas, in px.                                     |
| W      | The width of the control, in px.                                                   |
| H      | The height of the control, in px.                                                   |
| Angle   | The rotation angle of the control                                                         |
| Opacity | The opacity of the control                                                           |
| Corner Radius   | The arc radius of the corners around the control                                                   |
| Fill   | The fill color of the control<br />The fill degree of the control<br />The direction of the fill<br />Set the range of fill values |
| Border   | The border color of the control<br />The line type of the control's border<br />The line thickness of the control's border     |
| Permission   | Set the permissions of the control, control display/hide or enable/disable                                  |
