## 1. Overview

Real-time trend chart controls are used to visually display the dynamic trends of data changes over time. Through the coordinate system of the time axis (X-axis) and data axis (Y-axis), they reflect the fluctuation patterns and development trends of data in real-time.

![Image Description](../../images/控件照片/实时趋势/1.png)

## 2. Usage Scenarios

Real-time trend controls are used in the following scenarios:

### 1. Real-time Monitoring Scenarios

- When real-time monitoring of dynamic data such as device operating parameters and process indicators is needed
- Trend monitoring of process parameters such as temperature, pressure, and flow in industrial automation systems
- Real-time tracking of system performance indicators and resource usage rates

## 3. Quick Start

1. Drag the real-time trend chart control to the canvas and adjust position and size
2. Configure curve variables, names, and display styles in "Data" to define trend chart Y-axis data
3. Set the "Query" time range to define the data display interval
4. Configure the display styles and scale settings of the X-axis and Y-axis
5. Adjust color scheme (background, grid, axis lines, etc.)
6. Set legend position and display control
7. Configure refresh frequency to ensure real-time data updates

1. Drag the real-time trend control to the canvas and adjust position and size
2. Add data rows in Data for Y-axis data in the chart. Data rows can bind data that users want to focus on, and you can set the line style of this data
3. You can set the chart's refresh frequency, as well as the X-axis time range and time format
