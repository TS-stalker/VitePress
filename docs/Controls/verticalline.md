## 一、概述

垂直线形状控件用于在画布上创建和编辑垂直方向的直线图形元素，专为需要严格垂直对齐和分隔的设计场景提供精准工具，确保线条的垂直和精确布局。

![图片描述](../images/控件照片/垂直线/1.png)

## 二、使用场景

垂直线控件在以下场景中使用：

### 1.数据可视化场景

- 用于图形化展示数据关系的辅助形状
- 数据分布图、关系图中的垂直线元素

## 三、快速开始

1. 从工具栏选择垂直线控件
2. 将鼠标放置在画布起始位置，按住鼠标左键
3. 垂直拖动鼠标调整直线长度
4. 释放鼠标左键完成垂直线创建（自动保持垂直）
5. 通过选中框的端点控制点调整直线长度
6. 使用中间控制点调整直线水平位置
7. 在属性面板中精确设置长度、位置和样式参数

## 四、属性详解

| 名称   | 描述                                                   |
| ------ | ------------------------------------------------------ |
| 名字   | 此控件的名称。                                         |
| X      | 控件左侧距画布左侧的距离，单位px。                     |
| Y      | 控件顶部距画布顶部的距离，单位px。                     |
| W      | 控件的宽度，单位px。固定为1px                          |
| H      | 控件的高度，单位px。                                   |
| 角度   | 控件的旋转角度                                         |
| 透明度 | 控件的透明度                                           |
| 线条   | 控件的线条颜色<br />控件的线条样式<br />控件的线条粗细 |
| 权限   | 设置控件的权限，控件显示隐藏/启动禁用                  |
