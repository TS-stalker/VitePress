## 一、概述

滑动条控件用于在指定数值范围内可视化地调整和选择参数值，通过滑动操作实现精确的数值输入和实时反馈。

![图片描述](../images/控件照片/滑动条/1.png)

## 二、使用场景

滑动条控件在以下场景中使用：

### 1.参数调节场景

- 当需要在连续数值范围内精确调整参数时
- 音量控制、亮度调节、速度设置等连续变量调整
- 实时预览参数变化效果的交互场景
- 配合其他控件实现联动的参数调整
- 实时监控和控制系统参数

## 三、快速开始

1. 拖拽滑动条控件到画布并调整位置尺寸
2. 设置"最小值"和"最大值"定义数值范围
3. 配置"值"属性设置初始位置
4. 调整样式属性（背景色、滑块颜色等）
5. 配置提示信息显示样式和小数位精度
6. 设置"动作"定义数值变化时的交互行为

## 四、属性详解

| 名称     | 描述                                                                                                                                                                                                                                                                         |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 名字     | 此控件的名称。                                                                                                                                                                                                                                                               |
| X        | 控件左侧距画布左侧的距离，单位px。                                                                                                                                                                                                                                           |
| Y        | 控件顶部距画布顶部的距离，单位px。                                                                                                                                                                                                                                           |
| W        | 控件的宽度，单位px。                                                                                                                                                                                                                                                         |
| H        | 控件的高度，单位px。                                                                                                                                                                                                                                                         |
| 角度     | 控件的旋转角度                                                                                                                                                                                                                                                               |
| 透明度   | 控件的透明度                                                                                                                                                                                                                                                                 |
| 显示     | 背景<br />设置整个背景条的背景色。<br />滑块颜色<br />设置滑块部分颜色。<br />最小值<br />设置滑动条的最小值。<br />最大值<br />设置滑动条的最大值。<br />值<br />设置滑动条的当前值。说明：一旦绑定了“值”，则在运行页面滑块不可拖动。                                       |
| 提示信息 | 设置滑动块提示信息的显示样式。<br />显示提示<br />控制提示信息的显示或隐藏。<br />背景<br />设置提示信息的背景色。<br />边框色<br />设置提示信息的边框颜色。<br />小数位<br />设置提示数值的小数位。<br />字体<br />设置提示信息的字体型号、字体大小、粗体、斜体、字体颜色。 |
| 权限     | 设置控件的权限，控件显示隐藏/启动禁用                                                                                                                                                                                                                                        |
