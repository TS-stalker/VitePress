## 一、概述

报警列表控件用于集中显示实时和历史报警信息，支持多节点报警数据整合、实时状态监控和历史记录查询，提供报警确认、搁置、筛选等管理功能，确保及时有效的报警响应和处理。

![图片描述](../images/控件照片/报警列表/1.png)

## 二、使用场景

报警列表控件在以下场景中使用：

### 1.故障处理场景

- 需要快速定位和确认当前系统异常状态时
- 设备故障报警的处理跟踪和状态更新
- 报警确认、搁置、恢复等操作管理

## 三、快速开始

1. 拖拽报警列表控件到画布并调整位置尺寸
2. 设置"显示"属性配置需要展示的功能按钮
3. 配置"闪烁"频率设置报警项的视觉提示效果
4. 设置"自动刷新"时间间隔确保数据实时更新
5. 调整边框样式和表格线颜色
6. 配置"表头"和"表体"的样式和字体
7. 设置"筛选面板"样式优化查询体验
8. 在"其他"中设置报警最多显示条数

## 五、属性详解

| 名称                              | 描述                                                                     |
| --------------------------------- | ------------------------------------------------------------------------ |
| 名字                              | 此控件的名称。                                                           |
| X                                 | 控件左侧距画布左侧的距离，单位px。                                       |
| Y                                 | 控件顶部距画布顶部的距离，单位px。                                       |
| W                                 | 控件的宽度，单位px。                                                     |
| H                                 | 控件的高度，单位px。                                                     |
| ![图片描述](../images/透明度.png) | 控件的透明度                                                             |
| 显示                              | 设置控件上显示哪些按钮。                                                 |
| 闪烁                              | 设置报警闪烁的频率以及是否启用闪烁                                       |
| 自动刷新                          | 设置自动刷新报警列表的时间间隔                                           |
| 边框颜色                          | 设置控件外边框及表体的表格线颜色。                                       |
| 边框粗细                          | 设置控件外边框及表体的表格线的粗细。                                     |
| 表头                              | 设置表头的背景色、字体型号、字体大小、加粗、倾斜、字体颜色。             |
| 表体                              | 设置表体的背景色、字体型号、字体大小、加粗、倾斜、字体颜色。             |
| 筛选面板                          | 设置筛选面板的样式。<br />![图片描述](../images/控件照片/报警列表/4.png) |
| 其他                              | 设置报警最多显示条数                                                     |
| 权限                              | 设置控件的权限，控件显示隐藏/启动禁用                                    |
