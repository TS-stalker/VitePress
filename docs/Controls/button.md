## 一、概述

按钮控件用于触发特定操作或显示状态信息，通过点击、悬停等交互方式实现用户与系统的指令传递，支持丰富的视觉效果和状态反馈。

![图片描述](../images/控件照片/按钮/1.png)

## 二、使用场景

按钮控件在以下场景中使用：

### 1.操作触发场景

- 当需要用户确认提交、保存数据或执行特定命令时
- 系统功能的手动触发和控制指令下发
- 设备启停状态、连接状态、报警状态的可视化显示

### 2.导航跳转场景

- 界面间的跳转和功能模块的切换
- 多步骤流程中的前进和后退控制

## 三、快速开始

1. 拖拽按钮控件到画布并调整位置尺寸
2. 在"填充文本"属性中设置按钮显示的文字内容
3. 配置"颜色"属性定义不同交互状态下的视觉效果
4. 设置边框粗细和圆角曲率调整按钮外观样式
5. 可选择添加"填充背景图"增强视觉效果
6. 配置字体样式和对齐方式优化文本显示
7. 设置"动作"定义按钮点击后的交互行为

## 四、属性详解

| 名称                            | 描述                                                                                                                                                               |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 名字                            | 此控件的名称。                                                                                                                                                     |
| X                               | 控件左侧距画布左侧的距离，单位px。                                                                                                                                 |
| Y                               | 控件顶部距画布顶部的距离，单位px。                                                                                                                                 |
| W                               | 控件的宽度，单位px。                                                                                                                                               |
| H                               | 控件的高度，单位px。                                                                                                                                               |
| ![图片描述](../images/圆角.png) | 按钮四个角的圆角曲率。                                                                                                                                             |
| 填充文本                        | 控件显示的内容。<br />![图片描述](../images/控件照片/按钮/4.png)                                                                                                   |
| 填充背景图                      | 控件的背景图，仅支持JPG、GIF、PNG、SVG、JPEG格式。                                                                                                                 |
| 边框粗细                        | 边框的线条粗细。                                                                                                                                                   |
| 颜色                            | 设置控件不同操作状态下的颜色效果。状态包括：默认、悬浮、按下。<br />可以为每种状态设置背景色、边框颜色、字体颜色。<br />![图片描述](../images/控件照片/按钮/5.png) |
| 字体                            | 设置文本内容的字体。包括字体型号、字体大小、加粗、倾斜、水平对齐方式、垂直对齐方式。                                                                               |
| 权限                            | 设置控件的权限，控件显示隐藏/启动禁用                                                                                                                              |
