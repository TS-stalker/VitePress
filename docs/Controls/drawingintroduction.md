## 一、概述

DipuOne 内置了一套功能完善的绘图工具。利用这些工具，您可以直接在画布上绘制和编辑基础的几何图形与自定义形状，从而无需依赖外部图片即可丰富界面的视觉元素。



## 二、绘图控件一览

系统提供的绘图控件包括：

| 控件名称 | 主要功能                       |
| ---- | -------------------------- |
| 折线   | 绘制由多个线段连接而成的折线路径。          |
| 多边形  | 绘制由多条边组成的闭合多边形（如三角形、五边形等）。 |
| 管道   | 绘制用于表示流程或连接的管道形状。          |

这些绘图控件创建的图形，与常规控件一样，**支持属性绑定、动画和动作配置**，可以完美地融入您的交互式应用设计中。
