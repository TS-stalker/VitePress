## 一、概述

折线形状控件用于在画布上创建和编辑由多个线段连接而成的折线图形，支持自由绘制和精确编辑，适用于创建复杂路径、连接线和多边形轮廓。

![图片描述](../images/控件照片/折线/1.png)

## 二、使用场景

折线控件在以下场景中使用：

### 1.数据可视化场景

- 用于绘制多边形轮廓、自定义形状和复杂路径
- 折线图的基础线条、流程图中的自定义连接路径
- 当需要创建自定义趋势线、连接线或路径时

## 三、快速开始

1. 单击鼠标左键，从工具栏选择折线控件
2. 在画布起始位置单击鼠标左键并拖动鼠标绘制第一条线段
3. 单机鼠标左键完成第一条线段，再次按下拖动绘制下一条线段
4. 重复步骤绘制多个线段，单击鼠标右键确定终点位置
5. 双击折线进入编辑状态，调整各个顶点位置
6. 在属性面板中精确设置样式和全局参数

## 四、属性详解

| 名称   | 描述                                                   |
| ------ | ------------------------------------------------------ |
| 名字   | 此控件的名称。                                         |
| X      | 控件左侧距画布左侧的距离，单位px。                     |
| Y      | 控件顶部距画布顶部的距离，单位px。                     |
| W      | 控件的宽度，单位px。固定为1px                          |
| H      | 控件的高度，单位px。                                   |
| 角度   | 控件的旋转角度                                         |
| 透明度 | 控件的透明度                                           |
| 线条   | 控件的线条颜色<br />控件的线条样式<br />控件的线条粗细 |
| 权限   | 设置控件的权限，控件显示隐藏/启动禁用                  |
