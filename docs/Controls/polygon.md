## 一、概述

多边形形状控件用于在画布上创建和编辑多边闭合图形，支持从三角形到复杂多边形的灵活绘制，通过顶点控制实现精确的图形设计和编辑。

![图片描述](../images/控件照片/多边形/1.png)

## 二、使用场景

多边形控件在以下场景中使用：

### 1.数据可视化场景

- 用于创建现代界面中的非标准形状元素
- 自定义图形符号和视觉标识
- 当需要创建三角形、五角星、六边形等标准几何图形时

## 三、快速开始

1. 单击鼠标左键从工具栏选择多边形控件
2. 在画布起始位置单击鼠标左键确定第一个顶点
3. 移动鼠标查看预览线条，再次单击鼠标左键确定第二个顶点
4. 继续移动并单击添加更多顶点，形成多边形轮廓
5. 单击鼠标右键完成多边形绘制（自动闭合）
6. 通过选中框的控制点调整整体大小和旋转角度
7. 双击多边形进入编辑状态，拖动顶点调整图形形状

1.2 实际组态教程：

1. 单击鼠标左键，从工具栏选择折线控件
2. 在画布起始位置单击鼠标左键并拖动鼠标绘制第一条线段
3. 单机鼠标左键完成第一条线段，再次按下拖动绘制下一条线段
4. 重复步骤绘制多个线段，单击鼠标右键确定终点位置

## 四、属性详解

| 名称   | 描述                                                                   |
| ------ | ---------------------------------------------------------------------- |
| 名字   | 此控件的名称。                                                         |
| X      | 控件左侧距画布左侧的距离，单位px。                                     |
| Y      | 控件顶部距画布顶部的距离，单位px。                                     |
| W      | 控件的宽度，单位px。                                                   |
| H      | 控件的高度，单位px。                                                   |
| 角度   | 控件的旋转角度                                                         |
| 透明度 | 控件的透明度                                                           |
| 填充   | 控件的填充色<br />控件的填充程度<br />填充的方向<br />设置填充值的范围 |
| 边框   | 控件的边框颜色<br />控件的边框的线条种类<br />控件的边框的线条粗细     |
| 权限   | 设置控件的权限，控件显示隐藏/启动禁用                                  |
