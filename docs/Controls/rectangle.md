## 一、概述

矩形形状控件用于在画布上创建和编辑矩形图形元素，通过直观的拖拽操作和属性配置实现精确的图形设计，适用于界面布局、内容容器和视觉元素创建等多种应用场景。

![图片描述](../images/控件照片/矩形/1.png)

## 二、使用场景

矩形控件在以下场景中使用：

### 1.数据可视化场景

- 用于图形化展示数据关系的辅助形状
- 数据分布图、关系图中的矩形元素
- 进度指示、状态标识的矩形背景

## 三、快速开始

1. 从工具栏选择矩形控件
2. 将鼠标放置在画布目标位置，按住鼠标左键
3. 拖动鼠标调整矩形大小和比例
4. 释放鼠标左键完成矩形创建
5. 通过选中框的8个控制点调整矩形大小
6. 使用旋转控制点调整矩形角度
7. 在属性面板中精确设置尺寸、位置和样式参数

## 四、属性详解

| 名称   | 描述                                                                   |
| ------ | ---------------------------------------------------------------------- |
| 名字   | 此控件的名称。                                                         |
| X      | 控件左侧距画布左侧的距离，单位px。                                     |
| Y      | 控件顶部距画布顶部的距离，单位px。                                     |
| W      | 控件的宽度，单位px。                                                   |
| H      | 控件的高度，单位px。                                                   |
| 角度   | 控件的旋转角度                                                         |
| 透明度 | 控件的透明度                                                           |
| 填充   | 控件的填充色<br />控件的填充程度<br />填充的方向<br />设置填充值的范围 |
| 边框   | 控件的边框颜色<br />控件的边框的线条种类<br />控件的边框的线条粗细     |
| 权限   | 设置控件的权限，控件显示隐藏/启动禁用                                  |
