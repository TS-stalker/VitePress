## 一、概述

值显示控件用于显示文本内容，既支持静态文字展示，也可通过属性绑定动态数据源，实现内容的实时更新与动态显示。

![图片描述](../images/控件照片/值显示/1.png)

## 二、使用场景

值显示控件在以下场景中使用：

### 1.数据监控场景

- 当需要实时显示设备状态、传感器读数等动态数据时
- 工业监控系统中的温度、压力、流量等工艺参数显示
- 可以通过趋势弹框看到数据历史趋势图，更直观展示数据

## 三、快速开始

1. 拖拽值显示控件到画布并调整位置尺寸
2. 在"数据"属性中绑定要显示的数据源
3. 设置"文本"属性定义显示内容（支持静态文本和动态绑定）
4. 配置"单位"和"格式"属性优化数据显示效果
5. 开启"趋势弹框"并设置时间范围和采样方式
6. 调整样式属性（填充色、边框、字体等）

## 四、属性详解

| Y        | 控件顶部距画布顶部的距离，单位px。                                                                                                                                                                                                                                                                                                    |
| W        | 控件的宽度，单位px。                                                                                                                                                                                                                                                                                                                  |
| H        | 控件的高度，单位px。                                                                                                                                                                                                                                                                                                                  |
| 角度     | 控件的旋转角度                                                                                                                                                                                                                                                                                                                        |
| 透明度   | 控件的透明度                                                                                                                                                                                                                                                                                                                          |
| 数据     | 控件接收的数据。<br />文本<br />需要显示的内容。可以手动输入，也可以点击绑定按钮进行绑定。<br />单位<br />设置显示的数据的单位。可以手动输入，也可以点击绑定按钮进行绑定。<br />格式<br />期望显示的内容格式。可以手动输入，也可以点击设置按钮，双击选择需要的格式。<br />文本溢出<br />设置文本溢出时的处理，隐藏、换行、省略3种可选 |
