## 一、概述

DipuOne 内置了一套功能完善的绘制形状工具。利用这些工具，您可以直接在画布上绘制和编辑基础的几何图形与自定义形状，从而无需依赖外部图片即可丰富界面的视觉元素。



## 二、形状控件一览

系统提供的形状控件包括：

| 控件名称 | 主要功能           |
| ---- | -------------- |
| 椭圆   | 绘制椭圆形或正圆形。     |
| 矩形   | 绘制直角矩形或正方形。    |
| 圆角矩形 | 绘制带有可调节圆角的矩形。  |
| 直线   | 绘制任意角度和长度的直线段。 |
| 水平线  | 绘制严格保持水平方向的直线。 |
| 垂直线  | 绘制严格保持垂直方向的直线。 |

这些形状控件创建的图形，与常规控件一样，**支持属性绑定、动画和动作配置**，可以完美地融入您的交互式应用设计中。
