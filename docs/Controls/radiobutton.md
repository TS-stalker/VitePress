## 一、概述

单选按钮控件用于在多个互斥选项中选择唯一项，当用户选择某个选项时，其他选项自动取消选中，确保始终只有一个选项处于激活状态。

![图片描述](../images/控件照片/单选按钮/1.png)

## 二、使用场景

单选按钮控件在以下场景中使用：

### 1.单向选择场景

- 当需要用户从多个互斥选项中必须选择一个时
- 需要确保唯一性的配置项选择
- 筛选条件的单选过滤设置

## 三、快速开始

1. 拖拽单选按钮控件到画布并调整位置尺寸
2. 在"选项文本"中设置各选项的显示名称
3. 在"选项值"中配置各选项对应的数据值
4. 设置"选中文本"或"选中值"确定默认选中项
5. 选择"样式"布局方式（水平或垂直排列）
6. 调整填充色、边框色等视觉样式
7. 设置"动作"定义选项变化时的交互行为

## 四、属性详解

| 名称     | 描述                                                                                                                                                                 |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 名字     | 此控件的名称。                                                                                                                                                       |
| X        | 控件左侧距画布左侧的距离，单位px。                                                                                                                                   |
| Y        | 控件顶部距画布顶部的距离，单位px。                                                                                                                                   |
| W        | 控件的宽度，单位px。                                                                                                                                                 |
| H        | 控件的高度，单位px。                                                                                                                                                 |
| 角度     | 控件的旋转角度                                                                                                                                                       |
| 透明度   | 控件的透明度                                                                                                                                                         |
| 选项文本 | 单选按钮的选项的名称。此处的“名称”和“选中文本”相对应。<br />![图片描述](../images/控件照片/单选按钮/3.png)                                                           |
| 选项值   | 单选按钮的选项的值。此处的“名称”和“选中值”相对应。<br />![图片描述](../images/控件照片/单选按钮/4.png)                                                               |
| 选中文本 | 当前选项的对应文本内容。                                                                                                                                             |
| 选中值   | 当前选项的值。<br />说明：选中文本和选中值只能二选一，设置一个选项后，另一个选项处于禁用状态。只有将当前设置的选项值清空或者解除绑定后，另一个选项才变为可编辑状态。 |
| 样式     | 单选按钮的布局样式。包含水平和垂直。                                                                                                                                 |
| 填充色   | 单选按钮的填充色。<br />![图片描述](../images/控件照片/单选按钮/5.png)                                                                                               |
| 边框色   | 单选按钮的边框颜色。<br />![图片描述](../images/控件照片/单选按钮/6.png)                                                                                             |
| 字体     | 设置文本内容的字体。包括字体型号、字体大小、字体颜色、粗体、倾斜、下划线。                                                                                           |
| 权限     | 设置控件的权限，控件显示隐藏/启动禁用                                                                                                                                |
