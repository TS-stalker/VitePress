## 一、概述

变量列表控件用于集中展示多个变量的名称和当前值，支持实时数据更新和数值格式化，提供直观的数据监控界面，便于用户同时查看和管理多个相关变量。

![图片描述](../images/控件照片/变量列表/1.png)

## 二、使用场景

变量列表控件在以下场景中使用：

### 1.参数管理场景

- 需要对多个配置参数进行集中查看和对比时
- 系统配置参数列表、设备设置参数组、工艺控制参数集
- 批量参数的快速检查和调整参考

## 三、快速开始

1. 拖拽变量列表控件到画布并调整位置尺寸
2. 在"数据"中点击绑定按钮选择需要监控的变量
3. 对数值型变量设置显示的小数位数
4. 在"显示"中配置表头、网格线、趋势弹框等显示选项
5. 设置"边框"颜色和粗细定义控件外观
6. 配置"表头"和"表体"的样式、颜色和交互效果
7. 设置斑马纹、选中行背景等增强可读性

## 四、属性详解

| 名称                                | 描述                                                                                                                                                 |
| ----------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| 名字                                | 此控件的名称。                                                                                                                                       |
| X                                   | 控件左侧距画布左侧的距离，单位px。                                                                                                                   |
| Y                                   | 控件顶部距画布顶部的距离，单位px。                                                                                                                   |
| W                                   | 控件的宽度，单位px。                                                                                                                                 |
| H                                   | 控件的高度，单位px。                                                                                                                                 |
| ![图片描述](../images/旋转角度.png) | 控件的旋转角度                                                                                                                                       |
| ![图片描述](../images/透明度.png)   | 控件的透明度                                                                                                                                         |
| 数据                                | 点击绑定按钮为控件绑定数据。在选择变量窗口双击变量进行绑定。绑定变量后，如果绑定的变量数据类型是数值型，则可以点击设置按钮设置该变量显示的小数位数。 |
| 显示                                | 设置该控件表头、行网格、列网格、趋势弹框、时间范围、采样方法等。                                                                                     |
| 边框                                | 设置边框色和边框粗细。                                                                                                                               |
| 表头                                | 设置表头的背景色、字体型号、字体大小、加粗、倾斜、字体颜色。                                                                                         |
| 表体                                | 设置表体的行高、背景色、字体型号、字体大小、加粗、倾斜、字体颜色、选中字体颜色、悬浮字体颜色、斑马纹、选中行背景、悬浮行背景。                       |
| 权限                                | 设置控件的权限，控件显示隐藏/启动禁用                                                                                                                |
